# image-compress
核心技术栈 编程语言：Python（作为主要的开发语言）。 GUI 框架：CustomTkinter（用于构建现代化、支持深色/浅色主题的桌面用户界面，是基于 Python 内置 tkinter 的增强版）。 拖拽功能支持：tkinterdnd2（为应用程序提供文件拖拽到窗口内的功能原生支持）。 图像处理：Pillow (PIL)（Python 最主流的图像处理库，提供图像读取、缩放、压缩、格式转换等功能）。 数据处理：NumPy（常用于在图像处理过程中进行高效的多维数组或矩阵计算）。
Python 运行环境：
需要安装 Python 3.x（建议通常为 Python 3.8 或以上版本，以便良好兼容 customtkinter 和其他库）。
必须配置好全局环境变量，使得可以在终端调用 python 或 Windows 专用的 py 命令。
PIP 包管理器：用来安装运行项目前必需的第三方依赖。
依赖包安装：运行前需要通过 pip install -r requirements.txt 安装上述提到的核心库。
通过 python main.py 启动图片压缩工具的主界面。
